# Roboto Font Provenance

## Source

Downloaded from the official upstream release of the Roboto 3 classic
family published by Google Fonts:

- Repository: https://github.com/googlefonts/roboto-3-classic
- Release: v3.015
- Release asset: `Roboto_v3.015.zip`
- Files extracted from the `android/static/` directory inside that zip
- Specimen page: https://fonts.google.com/specimen/Roboto

## Download date

2026-04-11

## Weights bundled

Only static weights used by the Material 3 type scale are bundled:

| File                 | Weight | CSS name |
|----------------------|--------|----------|
| Roboto-Light.ttf     | 300    | Light    |
| Roboto-Regular.ttf   | 400    | Regular  |
| Roboto-Medium.ttf    | 500    | Medium   |
| Roboto-Bold.ttf      | 700    | Bold     |

Weight 600 (SemiBold) is intentionally omitted — it is not part of the
Material 3 type scale.
